# M1Ra3A1 - Elementos Básicos y Estructuras de Control

## Descripción General
Este proyecto web, titulado **M1Ra3A1**, ha sido desarrollado como parte de la formación técnica en el **Centro de Capacitación en Tecnologías de la Información y las Comunicaciones (TIC's)** de **INTECAP**. El objetivo principal es demostrar el dominio de la lógica de programación y la manipulación dinámica del contenido en el navegador.

El sitio integra 15 ejercicios prácticos divididos en dos áreas fundamentales:
1. **Estructuras de Control:** Implementación de algoritmos lógicos utilizando ciclos y condicionales para resolver problemas matemáticos y de procesamiento de datos.
2. **Funciones y Formularios:** Creación de una interfaz interactiva que utiliza funciones con parámetros para procesar información ingresada por el usuario, generando resultados dinámicos en tablas y alertas.

## Tecnologías Utilizadas

| Tecnología | Descripción de Uso |
| :--- | :--- |
| **HTML5** | Definición de la estructura semántica del sitio, formularios de captura y contenedores de resultados. |
| **CSS3 / Flexbox** | Aplicación de estilos visuales, maquetación responsiva y diseño de tablas para una presentación profesional. |
| **JavaScript (ES6+)** | Desarrollo de la lógica de programación, incluyendo estructuras de control (if, for, while) y funciones con parámetros. |
| **DOM Manipulation** | Interacción dinámica con los elementos del navegador para actualizar el contenido y mostrar resultados en tiempo real. |
